This diagram demonstrates how Amazon CloudFront allows public traffic to access private objects stored in an S3 bucket.

How it works:

  • User requests content from the CloudFront distribution URL
  • CloudFront acts as a CDN (Content Delivery Network) and caches content at edge locations
  • S3 Bucket remains private, with CloudFront using Origin Access Identity (OAI) to securely access objects

This architecture provides security, performance, and global content delivery without exposing the S3 bucket publicly.
